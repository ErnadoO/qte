Quick Title Edition

An extension for phpBB 3.1

View project on GitHub

Installation

You can install this extension on the latest version of phpBB 3.1 by doing the following:

  1. Download the extension by downloading the latest ZIP-archive of 2.0.x branch of its GitHub repository.
  2. Check out the existing of the folder /ext/ernadoo/qte/ in the root of your board folder. Create folders if necessary.
  3. Copy the contents of the downloaded qte-2.0.x folder to /ext/ernadoo/qte/.
  4. Navigate in the ACP to Customise -> Extension Management -> Manage extensions -> Quick Title Edition.
  5. Click Enable.

Update

  1. Download the updated extension.
  2. Navigate in the ACP to Customise -> Extension Management -> Manage extensions -> Quick Title Edition and click Disable.
  3. Remove the contents of the folder /ext/ernadoo/qte/
  4. Copy the contents of the downloaded qte-2.0.x folder to /ext/ernadoo/qte/.
  5. Navigate in the ACP to Customise -> Extension Management -> Manage extensions -> Quick Title Edition and click Enable.

Uninstallation

Navigate in the ACP to Customise -> Extension Management -> Manage extensions -> Quick Title Edition and click Disable.

For permanent uninstallation click Delete Data and then you can safely delete the /ext/ernadoo/qte/ folder.
