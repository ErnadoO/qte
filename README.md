## Installation

Clone into phpBB/ext/ernadoo/qte:

    git clone https://github.com/ErnadoO/qte.git phpBB/ext/ernadoo/qte

Go to "ACP" > "Customise" > "Extensions" and enable the "Quick Title Edition" extension.
